# Idea — Personal "life OS" / GTD weekly-review system

> Layer-A Idea node (durable · brand-free). Model: [`STRATEGIC-LANDSCAPE-MODEL.md`](../../STRATEGIC-LANDSCAPE-MODEL.md). **Bootstrap seed.**

- **Topic(s):** [personal-productivity-life-os](../TOPICS/personal-productivity-life-os.md) (+ [structured-documents-and-cards](../TOPICS/structured-documents-and-cards.md), [ai-agents-orchestration](../TOPICS/ai-agents-orchestration.md))
- **One-line:** A personal, single-user "organize your life" surface — low-friction NL capture (daily-notes journal), a forward-looking GPS-mapped daily itinerary (Agenda), and an agent-assisted GTD Weekly Review that compresses the 1–2-hour ritual to a ~20-minute confirm-and-decide.
- **Axes (Idea-level):** Conviction = **active** (Divia.Life is current-build, Phase 00) for the mobile surface; **aspirational** for the agent-assisted GTD layer · Horizon = mobile life-OS **now / v1**; GTD-review agent **later** · Provenance = John's portfolio vision; the GTD layer from [LATER-002 §5](../../../_backlog_TODOs/LATER-002-recurring-autonomous-agent-tasks.md) · Leverage = `.dvai` / DiviaCards; Logseq-style daily notes; the "forward-looking location history" (inverse of Google Location History) framing; the clerk/executive trust boundary.
- **Idea↔Idea edges:** `sibling-of` [household home-OS](household-home-os.md) (materially different customer/regime: personal single-user mobile vs. household self-hosted server) · `enabled-by` the recurring autonomous-agent capability (the GTD first-pass; see Notes) · relates-to [AI contacts / relationship manager](ai-contacts-relationship-manager.md) (the Friends-&-Family aggregator is the relationship-cadence idea on the consumer side)
- **Channel (Idea→Venture, time-bounded):** mobile surface → [Divia.Life](../PRODUCTS/DiviaLife/divialife.md) (🟠 Flutter Phase 00; `divialife-flutter` first, native Kotlin/Swift later — see [LATER-004](../../../_backlog_TODOs/LATER-004-mobile-cross-platform-strategy.md)); the GTD-review layer → [Divia.AI Enterprise](../PRODUCTS/DiviaAI/divia_ai-enterprise.md) + [Swarm](../PRODUCTS/DiviaAI/divia_ai-agentswarms.md) co-deployment (per LATER-002 §5)
- **Realized by (Build Lines, Layer B):** TBD.
- **Notes:** Divia.Life's four tabs = Agenda, Journal, Messages, Friends-&-Family (a configurable aggregated feed from Instagram/Threads/X/BlueSky via per-network OAuth — itself an implicit-data engine). Local-first, on-device source of truth. The GTD Weekly Review attacks GTD's #1 failure mode (skipping the review); the agent is the **clerk**, the human the **executive** — it never completes/deletes/commits, stays at autonomy L0/L1.
