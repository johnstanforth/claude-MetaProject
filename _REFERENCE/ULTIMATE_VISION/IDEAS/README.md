# IDEAS — the Layer-A idea nodes

Brand-free, **durable** Idea nodes for the KSVGPS Strategic Landscape (Module 1) — actionable concepts, each tagged to ≥1 [Topic](../TOPICS/), carrying their own research and a time-bounded **channel symlink** to whichever [Venture](../PRODUCTS/) currently embodies them. Ideas **persist when their channel detaches** (a paused/pivoted venture returns its Ideas here, reusable later). Model: [`STRATEGIC-LANDSCAPE-MODEL.md`](../../STRATEGIC-LANDSCAPE-MODEL.md).
