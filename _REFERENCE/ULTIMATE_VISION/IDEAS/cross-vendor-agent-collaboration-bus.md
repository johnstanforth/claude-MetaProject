# Idea — Cross-vendor agent collaboration bus

> Layer-A Idea node (durable · brand-free). Model: [`STRATEGIC-LANDSCAPE-MODEL.md`](../../STRATEGIC-LANDSCAPE-MODEL.md). **Bootstrap seed.**

- **Topic(s):** [ai-agents-orchestration](../TOPICS/ai-agents-orchestration.md), [developer-tooling-ai-coding](../TOPICS/developer-tooling-ai-coding.md).
- **One-line:** A structured substrate that lets AI agents **from different vendors** (Claude, Codex, Gemini…) plus a human coordinate, divide, review, and **decide** — a typed-envelope message bus with an engine-owned task FSM and an append-only event log, owning only the cross-vendor bridge (intra-vendor fan-out is delegated to each vendor's own subagents).
- **Axes (Idea-level):** Conviction = **active** (shipping at laptop scale; deep research done at server scale) · Horizon = laptop CollabPair **now / v1**, the next-gen `aixodev-collabs` engine **near→mid** (research complete, app unbuilt) · Provenance = built to fully replace the third-party screen-scraping `ensemble` engine; the R1–R18 design · Leverage = the typed envelope + task FSM + event log; reusable as the quality pattern for every recurring-agent story.
- **Idea↔Idea edges:** sibling-of / pairs-with [multi-agent dev-team coordination](multi-agent-dev-team-coordination.md) (collabs = how agents talk/decide; workgroups = how work is claimed/tracked) · *(same-idea-different-scale: aixocode's synchronous laptop CollabPair ≈ the server-scale next-gen bus — one customer-value, two scales)*.
- **Channel (Idea→Venture, time-bounded):** [AIXO.Dev Platform / ExoDev](../PRODUCTS/AIXO.Dev/aixodev-platform.md) (active) — laptop-side in `aixocode`, next-gen design home in `aixodev-collabs`.
- **Realized by (Build Lines, Layer B):** `aixocode` (native CollabPairs — 2 agents + human in a shared CHAT, control socket of 16 commands, guarded 3-phase `code_review_v1`; the data model already supports N → a future "CollabGroup") **and** `aixodev-collabs` (the planned next-gen Flask/SQLAlchemy engine: append-only event log + per-seat pull mailboxes, typed envelope, engine-owned task FSM, progress-ledger supervision, budget circuit-breakers). See the [engineering brief](../../SOFTWARE_DEV/aixodev-platform.md).
- **Notes:** A `[DEALBREAKER-HOOK]` (the irreversible substrate superseding the screen-scraping engine). Key research findings: **N=2 is the right scale**; **tests arbitrate, not rhetoric**; independent-answer-first + anonymized authorship beats forced-consensus debate; **cost is existential** (input tokens dominate ~150:1). aixocode's local collab subsystem is framed as the **laptop-scale precursor of Divia.AI AgentSwarms** — lessons flow aixocode → Swarm. **Cross-cluster note:** the general agent-collaboration substrate overlaps the Divia AgentSwarm cluster — flagged for dedup (this node is the dev-tooling-scoped view).
