# Idea — Hybrid deterministic + LLM workflow engine

> Layer-A Idea node (durable · brand-free). Model: [`STRATEGIC-LANDSCAPE-MODEL.md`](../../STRATEGIC-LANDSCAPE-MODEL.md). **Bootstrap seed.**

- **Topic(s):** [ai-agents-orchestration](../TOPICS/ai-agents-orchestration.md), [developer-tooling-ai-coding](../TOPICS/developer-tooling-ai-coding.md).
- **One-line:** Turn a development workflow from one giant natural-language prompt into a set of **explicitly typed steps** — each either an **LLM step** (dispatched to a model) or a **deterministic step** (Python code: a file-existence check, a min-length check, a validation suite exited 0) — so the hardest invariants become **code-enforced gates that cannot be skipped or hallucinated past**, with per-step model/effort tiering for cost efficiency.
- **Axes (Idea-level):** Conviction = **planned scope / captured** (LATER-001 Part 2; already-planned scope for the workgroups prototype) · Horizon = **mid** (lands when `aixodev-workgroups` opens a research phase) · Provenance = John's observation that gates like "STOP if no approved `xp_` file" and "zero test failures" are today only LLM-compliance hopes · Leverage = the Python/Flask workgroups app; per-step routing (e.g. Haiku-medium for mechanical steps, Fable-xhigh for hard analysis).
- **Idea↔Idea edges:** belongs-to / pairs-with [multi-agent dev-team coordination](multi-agent-dev-team-coordination.md) (the workgroups "brain" is its home; carries the autonomy ladder) · related-to [context-preserving search](context-preserving-search.md) (a deterministic search step feeding an LLM is an instance of this hybrid).
- **Channel (Idea→Venture, time-bounded):** [AIXO.Dev Platform / ExoDev](../PRODUCTS/AIXO.Dev/aixodev-workgroups.md) (planned) — prototyped in `aixodev-workgroups`, ultimately re-merged into the upstream `aixodev-web` platform service.
- **Realized by (Build Lines, Layer B):** `aixodev-workgroups` (then `aixodev-web`) — fold into its Phase-00 scoping when the project's research phase opens. See the [engineering brief](../../SOFTWARE_DEV/aixodev-platform.md).
