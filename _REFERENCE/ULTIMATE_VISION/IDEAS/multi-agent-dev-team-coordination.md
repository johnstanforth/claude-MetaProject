# Idea — Multi-agent dev-team coordination ("Asana for agents")

> Layer-A Idea node (durable · brand-free). Model: [`STRATEGIC-LANDSCAPE-MODEL.md`](../../STRATEGIC-LANDSCAPE-MODEL.md). **Bootstrap seed.**

- **Topic(s):** [ai-agents-orchestration](../TOPICS/ai-agents-orchestration.md), [developer-tooling-ai-coding](../TOPICS/developer-tooling-ai-coding.md).
- **One-line:** Coordinate a fleet of AI agents as **full team members** — persistent named personalities with assigned, claimed, locked, handed-off, dependency-ordered, and tracked work — the asynchronous "dozens-of-agents, Asana-model" task queue alongside humans across hundreds of projects.
- **Axes (Idea-level):** Conviction = **active** vision, but coordination prototype is **early** (`aixodev-workgroups` Phase-00 pending, zero code) · Horizon = **near→mid** (gated behind the aixocode↔aixodev-web server integration) · Provenance = the "agents as colleagues" thesis + the named-agent cast · Leverage = the platform's persistent agent personalities (each with a self-editing `SOUL.md`, each on a different model engine); the workgroups task FSM.
- **Idea↔Idea edges:** depends-on [cross-vendor agent collaboration bus](cross-vendor-agent-collaboration-bus.md) (the messaging/decision substrate — collabs = how agents talk/decide, workgroups = how work is claimed/tracked; the two halves of the next-gen system) · related-to [software-dev knowledgebase / Ontology graph](software-dev-knowledgebase-ontology-graph.md) (recurring tasks become graph-aware) · `enables` [hybrid deterministic+LLM workflow engine](hybrid-deterministic-llm-workflow-engine.md) (the autonomy-ladder home).
- **Channel (Idea→Venture, time-bounded):** [AIXO.Dev Platform / ExoDev](../PRODUCTS/AIXO.Dev/aixodev-platform.md) (active) — the platform's server-side async coordination pattern.
- **Realized by (Build Lines, Layer B):** `aixodev-workgroups` (the task-definition/assignment "brain": claim/lock, hand-off, dependency ordering, conflict resolution; Flask system-of-record **+ FastAPI status sidecar** for heartbeats) → merges into `aixodev-web`. See the [engineering brief](../../SOFTWARE_DEV/aixodev-platform.md).
